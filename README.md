# SpillR

## Prerequisites

- [Node.js](https://nodejs.org/)
- [Expo Go](https://expo.dev/go) app on your phone

## Setup

```bash
npm install
```

## Running

```bash
npm start
```

Scan the QR code in the terminal with the Expo Go app.

### Platform-specific

```bash
npm run android   # Android emulator
npm run ios       # iOS simulator
npm run web       # Browser
```
